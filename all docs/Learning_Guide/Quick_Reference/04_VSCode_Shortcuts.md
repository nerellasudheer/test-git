## Quick Scenarios

### Accidentally closed file?
`Ctrl + Shift + T` - Reopens it!

### Code looks messy?
`Shift + Alt + F` - Format document

### Need to rename variable everywhere?
1. Select variable
2. `Ctrl + D` repeatedly to select all
3. Type new name

### Text too small?
`Ctrl + =` to zoom in, `Ctrl + 0` to reset

### Compare two files?
1. Open first file
2. `Ctrl + \` to split
3. Open second file in new pane
